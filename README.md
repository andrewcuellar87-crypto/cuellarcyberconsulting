# Cuellar Cyber Consulting Portfolio — Version 2

This directory contains an upgraded, separate Version 2 of the fictional Upwork cybersecurity consulting portfolio. Version 1 remains preserved at `C:\Users\andre\upwork-cybersecurity-portfolio`.

## Start Here
- Open `index.html` for the interactive homepage.
- Open `case-studies/*.html` for dedicated case studies.
- Use `pptx/Cuellar-Cyber-Consulting-Upwork-Portfolio-V2.pptx` as the standalone executive deck.
- Use `pdf/Cuellar-Cyber-Consulting-Portfolio-V2.pdf` as the PDF web portfolio export.

## Folder Structure
- `assets/css/` — editable CSS
- `assets/js/` — small filtering script
- `assets/svg/` — editable diagrams and mockups
- `assets/png/` — rendered visuals
- `assets/thumbnails/` — Upwork-ready thumbnails
- `case-studies/` — five dedicated HTML case-study pages
- `markdown/` — editable copy and documentation
- `pptx/` — PowerPoint Version 2
- `pdf/` — PDF exports
- `qa/` — QA report and screenshots
